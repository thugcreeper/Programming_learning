To indicate that a variable does not contain a value, you can assign the value null to it .

Number

數字類型包括整數和浮點數。例如：

let integer = 42;
let float = 3.14;

String

字符串類型包括字符序列。例如：

let message = "Hello, world!";

Boolean

布爾類型表示真或假。例如：

let isTrue = true;
let isFalse = false;

Object

對象類型表示複合值。例如：

let person = {
  name: "Alice",
  age: 30
};

Undefined

未定義類型表示變量未賦值。例如：

let x;

Null

空類型表示沒有值。例如：

let y = null;

BigInt

大整數類型表示任意精度的整數。例如：

let bigIntNumber = 1234567890123456789012345678901234567890n;

Symbol

符號類型表示唯一且不可改變的值，用於對象屬性的標識符。例如：

let symbol = Symbol("description");

isFinite()

checks whether a number is a finite, legal number.

Infinite (not finite) values are Infinity, -Infinity, or NaN

typeof()

returns the type of a variable or an expression

NaN

represents "Not a Number" value

Infinity

displays a number that exceeds the limit of a floating point number
